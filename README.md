# Student Management System (C++)

A **console-based Student Management System** built using **C++** that allows users to manage student records and store them in a **CSV file (Excel compatible)**.

---

## Features

- Add student details
- Store:
  - Roll Number
  - Name
  - 5 Subject Marks
  - Total Marks
  - Percentage
- Search student by Roll Number
- Display all students (current session)
- Save data to CSV file (Excel supported)
- Menu-driven console interface

---

## Technologies Used

- C++
- OOP Concepts
- STL (`vector`)
- File Handling (`fstream`)
- Git & GitHub
